Hospital Management System (HMS)

A web application developed using Agile methodologies to manage hospital records

Overview

A Hospital Management System is a comprehensive software solution designed to streamline and manage various administrative and operational functions of a healthcare facility. It enables efficient handling of patient records, appointments, and resource allocation, improving overall hospital efficiency and patient care.
Key Features


Admin Dashboard: Complete control over patient records, doctor schedules, and hospital resources.
Doctor Dashboard: Access to personal schedule, patient appointments, and prescription management.
Role-Based Access: Customized views and permissions for administrators, doctors, and staff.

Technologies Used

Frontend: React
Backend: Spring Boot
Database: MySQL
Authentication: Spring Security with JWT
